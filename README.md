# opini v2.7 - 全能修復與雲端同步版

由 **Ferz** 建立與維護 | [個人網站](https://eblog-ferz.lovable.app)

opini 是一個強大的瀏覽器擴充功能工具箱，旨在為您提供極致的網址管理、音樂收藏、AI 影片分析與日常計算功能。

## 🚀 v2.7 更新亮點
- **📋 新增剪貼板 (Clipboard)**：隨時儲存文字資訊，支援一鍵複製與管理。
- **💾 新增 OpiniSave**：整合影片保存技術，讓 YouTube 影片收藏更簡單。
- **🛡️ 核心修復**：
  - 修復了網址儲存與 OpiniMusic 的功能缺失。
  - 為所有清單加入了「刪除」按鈕。
  - 修正了登入驗證邏輯，現在需輸入正確帳密。
- **🎨 品牌升級**：更換為官方全新 Logo，擴充功能字體與官網全面同步。
- **☁️ GitHub 同步**：資料即時同步至 `opini-data` 儲存庫，永不遺失。

## 🛠️ 主要功能
- **網址儲存**：自定義名稱，快速保存當前分頁。
- **OpiniMusic**：支援 YouTube 注入按鈕，一鍵收藏音樂。
- **AI 總結**：在 YouTube 影片旁即時生成內容摘要。
- **計算機**：全功能數學運算工具。
- **雲端同步**：支援跨裝置資料同步。

## 📖 安裝教學
1. **下載**：從 [官方網站](https://lan2015se-collab.github.io/opini/) 下載 `opini_v2.7.zip`。
2. **解壓縮**：將 ZIP 檔案解壓縮為一個「一般資料夾」。
3. **載入**：在瀏覽器擴充功能頁面開啟「開發人員模式」，點擊「載入解壓縮的擴充功能」並選擇該資料夾。
4. **設定**：進入擴充功能的「GitHub 設定」貼入 Token 即可開啟雲端同步。
