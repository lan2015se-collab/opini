# opini - 專業工具箱 (v2.3)

## 專案簡介
**opini** v2.3 迎來了重大的介面翻新與 YouTube 深度整合。我們致力於提供最純粹、最專業的工具體驗。

## v2.3 重大更新

### 🌑 專業黑底白字介面 (New Style!)
*   **視覺風格**：全面改為黑色背景、白色文字，並套用專業的**標楷體**，提供極佳的閱讀對比度與質感。
*   **介面簡化**：移除冗餘的分析頁面與內建播放器，回歸最直覺的清單管理。

### 🔴 YouTube 深度整合 (New Feature!)
*   **一鍵新增**：現在在 YouTube 影片頁面，「訂閱」按鈕旁邊會出現 **「新增到 OpiniMusic」** 按鈕。
*   **自動抓取**：點擊按鈕即可自動提取影片標題與網址，並儲存至擴充功能清單中。
*   **快速跳轉**：在擴充功能內點擊歌曲名稱，直接回到 YouTube 播放。

### 🧮 全功能計算機
*   持續提供穩定的標準計算功能，並維持黑白對比的高識別度設計。

## 安裝說明
1.  下載最新的 `opini_v2.3.zip`。
2.  解壓縮後，在瀏覽器擴充功能頁面載入解壓後的「資料夾」。
3.  **注意**：安裝後請重新整理 YouTube 頁面，即可看到新增按鈕。
