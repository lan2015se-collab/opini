# opini - 網址備註與管理擴充功能

## 專案簡介

**opini** 是一個輕量級的瀏覽器擴充功能，旨在幫助您更有效地管理和組織您在網路上瀏覽的資訊。無論是重要的研究資料、有趣的部落格文章，還是任何您想稍後回顧的網頁，opini 都能讓您即時儲存當前網頁的網址，並為其添加自定義的名稱和詳細備註。告別雜亂的書籤，opini 讓您的網頁收藏變得井然有序、一目了然。

## 主要功能

*   **即時儲存**：一鍵快速儲存當前瀏覽頁面的網址。
*   **自定義名稱**：為儲存的網址設定一個易於識別的名稱，取代冗長的網頁標題。
*   **詳細備註**：為每個儲存的網址添加個人備註，記錄您的想法、重點或回顧提示。
*   **便捷管理**：在擴充功能彈出視窗中直接查看所有已儲存的網址清單，並可點擊名稱快速跳轉。
*   **本地儲存**：所有資料安全地儲存在您的瀏覽器本地，確保隱私與速度。
*   **一鍵刪除**：輕鬆移除不再需要的網址記錄。

## 安裝指南 (適用於 Chromium 核心瀏覽器，如 Edge, Chrome)

請按照以下步驟將 **opini** 擴充功能載入到您的瀏覽器中：

1.  **下載擴充功能**：點擊上方連結下載 `opini_v1.2.zip` 檔案，並將其解壓縮到您電腦上一個容易找到的**空資料夾**中（例如：`C:\opini_extension` 或 `~/Downloads/opini`）。請確保 `manifest.json` 檔案直接位於該資料夾的根目錄下。

2.  **開啟擴充功能管理頁面**：
    *   **Microsoft Edge**：在網址列輸入 `edge://extensions/` 並按 Enter。
    *   **Google Chrome**：在網址列輸入 `chrome://extensions/` 並按 Enter。

3.  **啟用開發人員模式**：在擴充功能管理頁面的右上角（或左下角，依瀏覽器版本而異），找到並開啟「**開發人員模式**」開關。

4.  **載入解壓縮的擴充功能**：
    *   開啟開發人員模式後，頁面頂部會出現一個「**載入解壓縮的擴充功能**」（或「載入未封裝項目」）按鈕，點擊它。
    *   在彈出的檔案選擇視窗中，選擇您在步驟 1 中解壓縮的 **opini** 擴充功能資料夾（而不是 `zip` 檔案本身）。

5.  **開始使用**：
    *   成功載入後，您應該會在瀏覽器工具列中看到 **opini** 的圖示（通常是一個拼圖形狀的圖示，點擊後可以將其釘選）。
    *   點擊 **opini** 圖示，彈出視窗將顯示當前網頁的資訊，您可以輸入自定義名稱和備註，然後點擊「儲存此網頁」。
    *   所有儲存的網址將會顯示在清單中，點擊名稱即可跳轉到該頁面。
