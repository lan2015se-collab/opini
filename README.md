# opini - 網址備註與管理擴充功能

## 專案簡介

**opini** 是一個輕量級的瀏覽器擴充功能，旨在幫助您更有效地管理和組織您在網路上瀏覽的資訊。無論是重要的研究資料、有趣的部落格文章，還是任何您想稍後回顧的網頁，opini 都能讓您即時儲存當前網頁的網址，並為其添加自定義的名稱和詳細備註。告別雜亂的書籤，opini 讓您的網頁收藏變得井然有序、一目了然。

## 主要功能

*   **即時儲存**：一鍵快速儲存當前瀏覽頁面的網址。
*   **自定義名稱**：為儲存的網址設定一個易於識別的名稱，取代冗長的網頁標題。
*   **詳細備註**：為每個儲存的網址添加個人備註，記錄您的想法、重點或回顧提示。
*   **便捷管理**：在擴充功能彈出視窗中直接查看所有已儲存的網址清單，並可點擊名稱快速跳轉。
*   **本地儲存**：所有資料安全地儲存在您的瀏覽器本地，確保隱私與速度。
*   **一鍵刪除**：輕鬆移除不再需要的網址記錄。

## 安裝指南 (適用於 Edge 與 Chrome)

請按照以下步驟將 **opini** 擴充功能載入到您的瀏覽器中：

### 第一步：下載並準備檔案
1.  點擊上方的「Download」按鈕下載 `opini_v1.2.zip`。
2.  將下載的檔案解壓縮到一個**全新的資料夾**（例如：桌面上的 `opini` 資料夾）。

### 第二步：開啟瀏覽器擴充功能管理
*   **Microsoft Edge**：在網址列輸入 `edge://extensions/` 並按 Enter。
*   **Google Chrome**：在網址列輸入 `chrome://extensions/` 並按 Enter。

### 第三步：載入擴充功能
1.  在頁面中開啟「**開發人員模式**」開關。
2.  點擊「**載入解壓縮的擴充功能**」（Edge）或「**載入未封裝項目**」（Chrome）。
3.  在彈出的視窗中，選擇您在第一步中**解壓縮後的那個資料夾**。

### 第四步：開始使用
1.  點擊瀏覽器右上角的「擴充功能」圖示（拼圖形狀）。
2.  找到 **opini** 並將其釘選到工具列。
3.  現在，當您在任何網頁點擊 **opini** 圖示，就可以輸入自定義名稱與備註並儲存了！
