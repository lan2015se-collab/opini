# opini - 綜合網址與音樂管理工具 (v2.0)

## 專案簡介

**opini** 是一款強大的瀏覽器擴充功能，在 v2.0 版本中，我們從單一的網址儲存工具進化為多功能的**工具箱**。除了原有的網址備註功能外，新增了專為音樂愛好者設計的 **OpiniMusic**，讓您能輕鬆解析並管理來自 YouTube、Spotify 及 Apple Music 的音樂收藏。

## v2.0 全新功能

### 📌 網址儲存工具
*   **自定義名稱**：儲存網址時可自由設定顯示名稱。
*   **即時備註**：為網頁添加心得或重要資訊。
*   **一鍵跳轉**：點擊清單名稱直接開啟網頁。

### 🎵 OpiniMusic (新功能！)
*   **跨平台解析**：貼上 YouTube、Spotify 或 Apple Music 網址，自動解析歌名、歌手及發行資訊。
*   **YouTube 嵌入播放**：清單中提供 **PLAY** 按鈕，直接在擴充功能內嵌入播放 YouTube 音樂，無需切換分頁。
*   **快速導航**：
    *   **PLAY**：在彈出視窗內即時播放。
    *   **GO**：直接前往原始音樂網址。
    *   **返回**：播放時隨時返回歌曲管理清單。
*   **詳細資訊顯示**：自動顯示歌手、發佈者及所屬娛樂公司資訊。

## 下載與安裝

您可以點擊下方按鈕下載最新版本：

[![Download opini](https://img.shields.io/badge/Download-opini_v2.0-blue?style=for-the-badge&logo=github)](https://github.com/lan2015se-collab/opini/releases/download/v2.0/opini_v2.0.zip)

### 安裝步驟
1.  下載並解壓縮 `opini_v2.0.zip`。
2.  開啟瀏覽器擴充功能頁面 (`edge://extensions/` 或 `chrome://extensions/`)。
3.  開啟「**開發人員模式**」。
4.  點擊「**載入解壓縮的擴充功能**」，選擇解壓縮後的資料夾即可使用。
