# opini - 智能工具箱 (v2.4)

## 專案簡介
**opini** v2.4 導入了強大的 AI 影片總結功能，讓您在觀看 YouTube 時能快速獲取影片核心資訊。

## v2.4 全新功能

### 🤖 AI 影片總結 (New!)
*   **一鍵總結**：在 YouTube 影片頁面點擊「AI 總結」按鈕。
*   **智能分析**：自動分析影片發布者、時長及內容摘要，幫助您快速過濾內容。
*   **深度整合**：與「新增到 OpiniMusic」按鈕並列，打造最強 YouTube 助手。

### 🎨 官網風格優化
*   **現代感字體**：官方網站字體優化為類似 Manus AI Logo 的現代無襯線風格。
*   **教學保留**：維持完整的四步驟安裝指南，確保新手也能輕鬆上手。

### 🛠️ 功能持續優化
*   **穩定性提升**：優化了背景腳本 (Background Service Worker) 的執行效率。
*   **黑白風格維持**：擴充功能介面持續採用經典的黑底白字標楷體風格。

## 安裝說明
1.  下載最新的 `opini_v2.4.zip`。
2.  解壓縮後，在瀏覽器擴充功能頁面載入解壓後的「資料夾」。
