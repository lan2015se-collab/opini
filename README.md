# opini - 綜合工具箱 (v2.2)

## 專案簡介
**opini** 是一個多功能瀏覽器擴充功能。在 v2.2 中，我們帶來了更專業的播放器體驗與強大的全功能計算機。

## v2.2 全新功能

### 📺 標準化音樂播放器 (New!)
*   **專業嵌入格式**：完全套用 YouTube 官方標準嵌入代碼。
*   **全功能支援**：支援加速器、自動播放、陀螺儀及畫中畫模式。
*   **流暢切換**：優化了 PLAY 與 GO 模式的切換邏輯。

### 🧮 全功能計算機 (Upgrade!)
*   **專業佈局**：新增百分比 (%)、正負號切換 (±) 及倒退鍵 (⌫)。
*   **邏輯優化**：徹底修復運算錯誤與 Error 顯示問題，支援連續運算與小數點精確處理。
*   **視覺升級**：採用深色模式專業設計，操作更直覺。

### 🛡️ 版本防護系統
*   **自動檢測**：隨時確保您使用的是最新版本，保障功能穩定性。

## 安裝說明
1.  下載最新的 `opini_v2.2.zip`。
2.  解壓縮後，在瀏覽器擴充功能頁面載入解壓後的「資料夾」。
