# opini - 綜合工具箱 (v2.1)

## 專案簡介
**opini** 是一個多功能瀏覽器擴充功能，旨在提升您的網頁瀏覽與多媒體管理體驗。在 v2.1 中，我們不僅修復了核心功能，更增加了自動化版本防護與實用工具。

## v2.1 全新功能

### 🛡️ 自動版本防護 (New!)
*   **即時檢查**：擴充功能啟動時會自動比對 GitHub 上的最新版本。
*   **強制更新提示**：若偵測到新版本，會彈出提示並鎖定功能，引導您下載最新版本，確保功能正常且安全。

### 🧮 計算機工具 (New!)
*   **快速運算**：在工具箱中新增標準計算機，支援加減乘除與清除功能，方便在瀏覽網頁時進行簡單計算。

### 🎵 OpiniMusic (優化與修復)
*   **播放器修復**：解決了 YouTube 嵌入播放器可能出現的「錯誤 153」問題。
*   **解析優化**：優化了背景解析邏輯，提升音樂資訊獲取的穩定性。
*   **PLAY / GO 雙模式**：
    *   **PLAY**：直接在彈出視窗內嵌入播放。
    *   **GO**：直接跳轉至原始音樂網址。

### 📌 網址儲存
*   **自定義名稱與備註**：持續提供穩定的網頁紀錄與備註功能。

## 安裝說明
1.  下載最新的 `opini_v2.1.zip`。
2.  解壓縮後，您會得到一個包含程式碼的資料夾。
3.  **重要**：請在瀏覽器的擴充功能管理頁面中，點擊「載入解壓縮的擴充功能」，並選擇該資料夾（而非 .zip 檔案）。
