📖 安裝與使用指南

1. 下載與解壓

下載 opini_v2.5.zip 並解壓縮至資料夾。

2. 載入擴充功能

在瀏覽器 extensions 頁面開啟「開發人員模式」，點擊「載入解壓縮的擴充功能」並選擇該資料夾。

3. 註冊與登入

首次使用請先註冊帳號，您的資料將自動加密同步至 GitHub 儲存庫。
